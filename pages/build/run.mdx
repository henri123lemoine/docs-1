## Setup

Clone repository:
```bash
git clone https://github.com/Zeroth-Robotics/OpenLCH
```

Create conda environment:
```bash
conda create -n openlch python=3.10
conda activate openlch
```

Install dependencies:
```bash
cd OpenLCH
pip install -r requirements.txt
```
